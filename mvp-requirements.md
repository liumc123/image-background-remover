# Image Background Remover MVP 需求文档

## 1. 项目概述

- **产品名称**：待定（建议：RmBg / 去背景）
- **核心功能**：上传图片，自动去除背景，支持下载透明背景原图
- **目标用户**：设计师、电商卖家、内容创作者
- **技术栈**：Cloudflare Workers + Remove.bg API + 前端页面

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传（jpg/png/webp） | P0 |
| 背景移除 | 调用 Remove.bg API 处理图片 | P0 |
| 结果预览 | 实时展示处理前后对比 | P0 |
| 图片下载 | 下载透明背景 PNG | P0 |
| 进度提示 | 上传/处理中展示 loading 状态 | P0 |

### 2.2 限制说明

- 单张图片最大 10MB
- 支持格式：JPG、PNG、WebP
- 免费用户每分钟限 5 次请求（防滥用）

---

## 3. 非功能需求

### 3.1 性能
- 图片处理时间 < 5秒（取决于网络和图片大小）
- 页面首屏加载 < 2秒

### 3.2 成本
- Remove.bg 免费额度：50次/月
- 超出付费：$0.09/图
- Cloudflare Workers：每天 100,000 次免费请求

### 3.3 安全
- 不存储任何图片（内存处理，即用即删）
- 可加 API Key 保护，防止被抓包调用

## 5. 技术方案

### 5.1 Cloudflare Worker

```
请求流程：
用户上传图片 → Worker 接收 → 转成 Base64 → 调用 Remove.bg API → 返回结果图 → 用户下载
```

### 5.2 API 接口

```
POST /remove-bg
Body: FormData { image: File }
Response: { result: Base64 PNG }
```

### 5.3 环境变量

```
REMOVE_BG_API_KEY=xxx  # Remove.bg API Key
```
